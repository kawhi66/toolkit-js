# `father`

## Usage

```
$ father -h

  Usage: father [options]

  Options:

    --transform         Skip bundle and transform src to lib (default: false)
    -t, --type          Use specified build type (cjs | esm | umd, default: cjs)
    -w, --watch         Watch building (default: false)
    -h, --help          Print help
    -v, --version       Print version
```
